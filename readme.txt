== Description ==

Simple apache log parser for Wordpress.
Monitor site usage by ip-address.
Adds a dashboard graph widget displaying the daily bandwidth usage.

If you have installed the WP-IP2Nation-Installer the stats will
automatically include an extra "country" table column.

Download WP-IP2Nation-Installer here:
http://wordpress.org/extend/plugins/wp-ip2nation-installer/

== Installation ==

1. Extract into the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress

If you have enabled caching just make sure wp-content/cache is writable by webserver.
